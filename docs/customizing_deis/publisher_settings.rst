Customizing publisher
=========================

Settings used by publisher
---------------------------
The following etcd keys are used by the publisher component.

=======================================      =======================================================
setting                                      description
=======================================      =======================================================
/deis/publisher/logLevel                     Log level (default: error) Valid options: debug, error.
=======================================      =======================================================
